# Учебный репозиторий: Разработка веб-страниц на языке разметки HTML5 с использованием каскадных таблиц стилей CSS3

## 🔹 Описание проекта
Данный репозиторий представляет собой консолидированный архив практических и домашних заданий, выполненных в процессе изучения фронтенд-разработки. Проекты направлены на освоение семантической верстки, стилизации интерфейсов и программирования клиентской логики без использования сторонних библиотек и фреймворков.

## 🔹 Технический стек
* **Разметка:** HTML5
* **Стилизация:** CSS3
* **Логика:** JavaScript

## 🔹 Структура репозитория
Архив разбит на независимые директории. Каждая папка представляет собой изолированный мини-проект.

## 🔹 Архитектурные особенности
* **Изоляция:** Каждый проект является автономным и содержит собственные таблицы стилей (в директориях `css/` или `styles/`) и скрипты.
* **Отсутствие внешних зависимостей:** Весь код написан на чистом стеке. Развертывание не требует использования пакетных менеджеров (npm), Node.js или сборщиков (Webpack/Vite).
* **Структура ресурсов:** Графические материалы (растровые изображения и SVG) локализованы в папках `assets/` внутри соответствующих проектов.

## 🔹 Развёртывание и запуск

Проект опубликован с помощью GitHub Pages и доступен для просмотра онлайн: [https://enqenqenqenqenq.github.io/html/](https://enqenqenqenqenq.github.io/html/)

**Локальный запуск:**
Развёртывание серверной среды не является обязательным. Для просмотра на локальной машине:
1. Клонируйте репозиторий.
2. Откройте директорию интересующего проекта.
3. Запустите файл `index.html` (или специфичный корневой файл, например, `card_game.html`) в браузере.
